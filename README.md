## Overview

**Pizza Order** is a small Windows Forms (.NET) application that allows users to customize and order a pizza through a simple and clean graphical interface. The app demonstrates basic WinForms controls, event handling, and UI layout.

The user can select the pizza size, crust type, toppings, and where to eat, then view a live order summary with the total price.

---

## Features

* Select **Pizza Size**: Small, Medium, or Large
* Choose **Crust Type**: Thin or Thick
* Pick multiple **Toppings** using checkboxes
* Choose **Where To Eat**: Eat In or Take Out
* Automatic **Order Summary** update
* **Total Price** calculation
* **Reset Form** button to restore default values

---

## Technologies Used

* C#
* .NET Framework
* Windows Forms

---

## Educational Purpose

This project is ideal for beginners to practice:

* WinForms layout design
* Using RadioButtons and CheckBoxes
* Control grouping (GroupBox)
* Event-driven programming
* Basic UI logic and formatting
